# Spotify Music Trend Analysis and Popularity Prediction

## 📊 Project Overview
This project analyzes Spotify data to understand music trends and predict song popularity using Machine Learning and Power BI.

## 🔧 Tools Used
- Python (Pandas, Scikit-learn)
- Power BI
- Matplotlib, Seaborn

## 📈 Features
- Data preprocessing & cleaning
- PCA & standardization
- Regression & classification models
- Statistical analysis (Z-test, T-test, ANOVA, Chi-square)
- Interactive dashboard

## 📊 Output
- Popularity prediction
- Trend analysis
- Visual dashboard insights
